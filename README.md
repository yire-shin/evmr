# evmr

<!-- badges: start -->

<!-- badges: end -->

`evmr` is an R package for **extreme value modeling for r-largest order
statistics**.

The package provides tools for modeling environmental and climate
extremes using r-largest order statistics. It aims to support parameter
estimation, return level estimation, and model diagnostics for extreme
value distributions commonly used in hydrology, climatology, and
environmental sciences.

## Features

The `evmr` package provides a unified framework for extreme value
analysis based on r-largest order statistics.

Key features include:

- Parameter estimation for extreme value models
- Return level estimation for extreme events
- Model diagnostics and goodness-of-fit assessment
- Applications to climate and hydrological extremes

## Installation

You can install the development version of `evmr` from GitHub:

``` r
# install.packages("remotes")
remotes::install_github("your_github_id/evmr")
```

Replace `"your_github_id"` with your GitHub username.

This example shows how to load the package.

